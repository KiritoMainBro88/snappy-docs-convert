# Demo Script

Short demo outline for `kmb file tools`.

## 30-Second Website Demo

English narration:

1. "This is kmb file tools, a free local Windows document converter."
2. "Downloads are official and free from GitHub Releases."
3. "Files are processed by the desktop app on your machine, not by the website."
4. "The site has dark mode, Vietnamese support, release links, source code, and Discord support."

Vietnamese narration:

1. "Đây là kmb file tools, công cụ chuyển đổi tài liệu Windows miễn phí."
2. "Bản chính thức tải miễn phí từ GitHub Releases."
3. "Tệp được xử lý trên máy của bạn, không tải lên website."
4. "Website có dark mode, tiếng Việt, link release, mã nguồn và Discord hỗ trợ."

## 60-Second Desktop Demo

English narration:

1. "Open the app. Engine cards show Word, PowerPoint, LibreOffice, and PDF renderer status."
2. "Add a PDF, choose Images, select an output folder, and start."
3. "The app creates page images locally."
4. "Open PDF Tools, choose merge or split, add demo files, and run."
5. "Unsupported files are reported clearly instead of silently ignored."
6. "Everything is local: no upload, no telemetry, no account."

Vietnamese narration:

1. "Mở app. Các thẻ engine hiển thị Word, PowerPoint, LibreOffice và PDF renderer."
2. "Thêm PDF, chọn xuất ảnh, chọn thư mục xuất rồi bắt đầu."
3. "App tạo ảnh từ trang PDF ngay trên máy."
4. "Mở Công cụ PDF, chọn gộp hoặc tách, thêm file demo rồi chạy."
5. "File không hỗ trợ được báo rõ, không bị bỏ qua âm thầm."
6. "Mọi thứ chạy cục bộ: không upload, không telemetry, không cần tài khoản."

## Key Points

- Local-only.
- No upload.
- Batch conversion.
- PDF tools.
- Free and open source.
- Discord support: https://discord.gg/kZ3U36ncun
