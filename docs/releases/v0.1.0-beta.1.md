# kmb file tools v0.1.0-beta.1

Status: Public beta / prerelease

## Highlights

- Local-only Windows document converter.
- Batch conversion.
- Office Word/RTF to PDF verified.
- PDF to PNG/JPEG verified.
- PDF Toolbox V1:
  - Merge PDFs
  - Split PDF
  - Extract pages
  - Rotate pages
  - Images to PDF
- English/Vietnamese UI.
- Portable ZIP.
- Installer EXE.
- Static frontend website.

## Known Limitations

- Unsigned app/installer may trigger Windows SmartScreen.
- Framework-dependent build requires .NET 9 Desktop Runtime on Windows x64.
- LibreOffice fallback requires user-installed LibreOffice; real LibreOffice smoke remains pending on this machine.
- PowerPoint real smoke fixture remains pending.
- No code signing yet.
- No MSI/MSIX installer yet.
