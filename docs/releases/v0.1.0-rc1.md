## Known Issues

- Unsigned executable may trigger Windows SmartScreen.
- Framework-dependent build may require .NET 9 Desktop Runtime.
- LibreOffice fallback requires user-installed LibreOffice.
- PDF to Images via GUI produces output but freezes the app during conversion.
- Unsupported file add does not show clear feedback.
- Public beta is blocked until GUI responsiveness/feedback issues are fixed or explicitly accepted as pre-release limitations.
- Final license must be approved before public source/release.
