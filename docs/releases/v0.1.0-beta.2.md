# kmb file tools v0.1.0-beta.2

Status: Public beta / prerelease

## Highlights

- App Settings page with Appearance, Language, Updates, and About sections.
- App dark mode: System, Light, and Dark.
- App language detection and persistence: Auto/System, English, Vietnamese.
- Manual update center V1:
  - checks GitHub Releases,
  - supports Stable and Prerelease/Beta channels,
  - downloads selected installer/ZIP only after user action,
  - verifies GitHub release asset SHA256 digest when available,
  - launches installer or opens the ZIP folder only after user confirmation.
- Website dark mode and browser language detection with persisted choices.
- Website demo screenshots/video section.
- OCR Lite roadmap docs for low-end Windows machines.

## Known Limitations

- The app is unsigned. Windows SmartScreen or Unknown Publisher warnings may appear.
- Update flow is manual check/download/install, not silent background auto-update.
- LibreOffice real smoke is still pending unless LibreOffice is installed locally.
- PowerPoint real smoke fixture is still pending.
- OCR is roadmap only; OCR is not implemented in this release.
- No code signing, MSI/MSIX, or Velopack updater yet.

## Privacy

- No cloud upload.
- No telemetry, analytics, tracking, or account requirement.
- Website is frontend-only and does not process files.

## Update Instructions

- Existing users should download the installer or portable ZIP from this GitHub Release.
- Future beta.2+ users can use Settings > Check for updates to detect newer releases.
