# Static Website Fallback

This folder contains a no-build static website skeleton for kmb file tools.

The newer polished frontend-only website lives under `website/` and uses Vite + React + TypeScript. Keep this folder as a simple fallback only.

Open locally:

```powershell
start .\website-static\index.html
```

Deploy target can be GitHub Pages or any static host. Download links point to GitHub Releases.

`assets/hero.png` and `assets/logo.png` are local bitmap assets. `favicon.ico` is local. No CDN, analytics, or remote assets are used.
